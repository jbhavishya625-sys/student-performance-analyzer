# 📊 StudentLens — Student Performance Analyzer

A **Data Visualization + LLM** project built with Python, Matplotlib, Seaborn, and Claude AI.

## 🗂 Project Structure
```
student-performance-analyzer/
├── student_performance_dataset.csv           # Raw data (120 students)
├── student_performance_cleaned.csv           # Cleaned data (auto-generated by Section 1)
├── Student_Performance_Analyzer.ipynb        # Main analysis notebook (all sections)
├── commit9_streamlit_charts.py               # Streamlit dashboard — all charts
├── commit10_streamlit_llm.py                 # Full Streamlit app: charts + LLM insights
├── requirements.txt
└── plots/                                    # Chart images saved by the notebook
    ├── 01_subject_scores.png
    ├── 02_grade_distribution.png
    ├── 03_correlation_analysis.png
    ├── 04_attendance_studyhours.png
    ├── 05_subject_comparison.png
    └── 06_top_bottom_students.png
```

## 📦 Installation
```bash
pip install -r requirements.txt
```

## 🔑 Set API Key
Before running Section 8 (LLM Insights) in the notebook, set your Anthropic API key:
```python
import os
os.environ["ANTHROPIC_API_KEY"] = "your-api-key-here"
```
Get your key at: https://console.anthropic.com

## 🚀 Run the Notebook
Open and run all cells in order:
```bash
jupyter notebook Student_Performance_Analyzer.ipynb
```

## 🚀 Run the Streamlit Dashboard
```bash
streamlit run commit10_streamlit_llm.py
```

## 📓 Notebook Sections

| # | Section | Description |
|---|---------|-------------|
| 1 | Data Loading & Cleaning | Load raw CSV, fix NaN, fix negative study hours, cap attendance at 100% |
| 2 | Subject-wise Score Visualization | Average bar chart + box plot per subject |
| 3 | Grade Distribution Analysis | Donut chart, count bar, attendance-band stacked bar |
| 4 | Correlation Analysis | Heatmap + scatter plots for study hours & attendance vs score |
| 5 | Attendance & Study Hours Deep Dive | Histograms by grade, bubble chart, grouped bar |
| 6 | Subject Comparison per Grade | Radar chart + violin plots per subject & grade |
| 7 | Top vs Bottom Performer Analysis | Top 10 vs Bottom 10 student comparison |
| 8 | LLM Integration — Claude AI Insights | 4 AI-generated insight types via Claude API |

## 📌 Dataset Features
| Column | Description |
|--------|-------------|
| Student_ID | Unique student identifier |
| Math, Physics, Chemistry, English, Computer_Science | Subject scores (0–100) |
| Attendance_Percent | Class attendance percentage |
| Study_Hours_Per_Day | Daily study hours |
| Average_Marks | Overall average score |
| Final_Grade | A / B / C / D |

## 🤖 LLM Insight Types
| Insight | Description |
|---------|-------------|
| 📋 Class Performance Summary | 5-point summary of overall class trends |
| 💡 Teacher Recommendations | 5 data-driven suggestions to improve outcomes |
| ⚠️ At-Risk Student Profile | Early warning indicators for struggling students |
| 📚 Subject Improvement Focus | Which subjects need attention and how to improve |
